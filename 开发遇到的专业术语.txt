i18n:internationalization的简写。国际化的意思

DAO:数据访问对象(Data Access Object)的缩写， java开发中专门用来对数据库的数据作存取操作

DTO:数据传输对象(Data Transfer Object)的缩写

HTTP:HyperText Transfer Protocol的缩写，是互联网最为广泛的一种协议，总之你要上网就要同意这个协议

HTML:超文本标记语言(Hyper text MarkUp Language)的缩写， 专门用来显示网页上的数据

XML:可扩展性标记语言(Extensible MarkUp Language)的缩写，常用作一些配置文件。HTML语言是XML语言的一个子类

URL:统一资源定位符(Uniform Resource Locator)的缩写，一个URL所代表的地址为可访问的地址

URI:统一资源标识符(Uniform Resource Identification)的缩写。
	它包括URL，但是范围比URL大。因为一个URI可以是一个虚假的地址

MIME:多用途互联网邮件扩展类型(Multipurpose Internet Mail Extensions)的缩写

JSP：java服务页面(Java Server Pages),是一种动态网页

SSH:java web三大框架的缩写，他们分别是(Struts2,Hibernate,Spring)

HQL:java web框架Hibernate自带的数据库查询语言Hibernate Query Language(速度没有SQL快)

SQL:结构查询语言(Structure Query Language)的缩写，对数据库的数据进行操作的语言
    SQL 又分为DML与DDL
	DDL：Data Definition Language 数据定义语言，定义表结构等
	DML：Data Manipulation Language 数据操纵语言，例如SELECT、UPDATE、DELETE语句

ORM:对象-关系映射(Object-Relation Mapping)的缩写

DOM:文档对象模型(Document Object Model)的缩写，是一种解析XML文件的思想

DOM4J:java的XML处理方式(Document Object Model for Java)

AJAX:异步的JavaScript和XML(Asynchronous JavaScript And XML)的缩写，
	   是一种可以不刷新页面就能与服务器进行数据量较小的交互的技术
	   
MVC:Model View Controller的缩写，是一种思想，上文的SSH框架就是基于MVC的思想

BOM:浏览器对象模型(Browser Object Model)的缩写。

SVN:Subversion的简称，是一个开源的源码控制系统。是CVS的替代版本

JSTL: Java Standard Tag Library  java标准标签库

OCP: Open Close Principle 设计模式七原则，对扩展开放，对修改关闭

GOF: Gang of Four(GOF) Design Patterns(设计模式)的作者(4人)的简称，现为设计模式的简称吧

UML:unified Model Language(统一建模语言)

RBAC: Role Base Access Control(基于角色的访问控制)
      现有RBAC新解：Resource Base Access Control 基于资源的控制访问，跳过了角色中间关系

ACID:Atomic(原子性)、Consistent(一致性)、Isolated(独立性)、Durable(持久性)

DTD:Document Type Definition(文档类型定义)

XSD:XML Schema Definition(XML的协议定义)

SOAP:Simple Object Access Protocol 简单对象访问协议，是webService的核心之一

SAAJ：SOAP Attach API for Java

OOP:Object-Oriented Programming 面向对象程序设计

SPA：Singleton Page Application 单页应用：所有跳转全部在一个页面
